# SDFM 2520

Premium streetwear e-commerce built with Next.js and TypeScript.

[Live Demo](https://v0-wstreetwear.vercel.app/)

## Stack

- Next.js 14
- TypeScript
- Tailwind CSS
- Vercel

## Getting Started

```bash
git clone https://github.com/LilDre7/v0-wstreetwear.git
cd v0-wstreetwear
pnpm install
pnpm dev
```

Open [http://localhost:3000](http://localhost:3000)

## Project Structure

```
app/          # Next.js app router
components/   # React components
styles/       # Global styles
public/       # Static assets
types/        # TypeScript types
```
