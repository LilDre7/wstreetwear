## About

SDFM 2520 is a curated streetwear collection showcasing premium urban fashion. The platform combines modern web technologies with a clean, responsive interface to deliver an optimal shopping experience across all devices.

**Product Range**: Classic Black ($150) 路 Premium Gray ($155) 路 Signature Navy ($160) 路 Limited Edition ($200)

## Tech Stack

- **Next.js 14** - React framework with App Router
- **TypeScript** - Type-safe development (92.8%)
- **Tailwind CSS** - Utility-first styling
- **Vercel** - Deployment and hosting

## Quick Start

```bash
git clone https://github.com/LilDre7/v0-wstreetwear.git
cd v0-wstreetwear
pnpm install
pnpm dev
```

Open [http://localhost:3000](http://localhost:3000) to view the app.

## Project Structure

```
app/          # Next.js app router

styles/       # Global styles

```
